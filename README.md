# E-Commerce App using ASP.NET Core MVC


An experimental online shopping cart project using C# Asp.NET Core,  MVC, SQLServer, HTML5, CSS3 and Bootstrap.


## Installation
* Latest Visual Studio 2019 version => 16.8 Or Visual Studio Code with C# Extesions
* dotnet SDK 3.1
* Sql Server 


## Technologies and Frameworks used
* ASP.NET MVC CORE 3.1
* Entity Framework Core 3.1
* ASP.NET Core Identity
* Bootstrap 4

## Application information
This is an online Game Store and it includes the following features:
* Online product catalog
* Product categories
* A Shopping cart
* A checkout page
* An administrative section to manage the catalog
